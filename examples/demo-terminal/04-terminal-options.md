# Terminal Options

### Syntax

```markdown
<!-- term: ./path -->
<!-- term: ./path :light -->
<!-- term: ./path | command -->
<!-- term: ./path :light | command -->
```

<!-- pause -->

### Features

- Real PTY connected to your shell
- ANSI color support (256 + truecolor)
- Dark and light themes
- macOS-style window chrome
- `Cmd +` / `Cmd -` to resize font
- Green button expands to fullscreen
- Yellow button restores
