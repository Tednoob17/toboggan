# What is this?

Toboggan can now embed **live terminals** inside slides.

Use the `<!-- term: path -->` comment in your markdown:

```markdown
<!-- term: ./my-project -->
```

<!-- pause -->

Features:
- Real PTY connected to your shell
- ANSI color support (256 + truecolor)
- Dark and light themes
- macOS-style window chrome
