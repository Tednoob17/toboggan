# Slide Styling

### Frontmatter classes

```toml
+++
classes = ["no_title", "wide", "cover", "center"]
+++
```

<!-- pause -->

### Available classes

| Class | Effect |
|---|---|
| `no_title` | Hides the slide title |
| `wide` | Expands to 95% viewport width |
| `cover` | Centers content (for title slides) |
| `center` | Centers article content |
| `spread-steps` | Distributes steps evenly |
